# Savora visual system

Savora is designed as a warm editorial kitchen companion: calm enough to use with messy hands, distinctive enough not to resemble a generic Material demo.

## Palette

- Herb green `#315C4C`: primary actions and identity
- Paprika `#C8583A`: one controlled accent for saved and highlighted states
- Warm cream `#F8F3E9`: daylight background
- Paper `#FFFCF6`: content surfaces
- Deep ink `#19332B`: high-contrast hero surfaces

## Type and hierarchy

Editorial headings use a serif system face where available. Interface and measurement text use the platform sans-serif for legibility. Large headings establish the first reading target; metadata stays compact and secondary.

## Layout

- 8-point spacing rhythm
- 44–50 logical-pixel minimum interactive targets
- Mobile bottom navigation; desktop navigation rail
- One-column mobile recipe cards, scaling to four columns on wide desktop
- Detail layouts split ingredients and method only when adequate width exists

## Imagery

Food photography is content, not decoration. Every catalogue card and detail hero uses a real dish image. Failures degrade to a semantic food placeholder without breaking layout.

## Accessibility

Icons with actions have tooltips, food images expose semantic labels, progress indicators expose intent, text remains dynamic, and color is never the sole carrier of selection or status.
